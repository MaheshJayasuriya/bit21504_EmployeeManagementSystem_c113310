How to run (short)

  1. Backend: open `backend/EmployeeManagementSystem` and run `dotnet run`. By default in Development the API listens on http://localhost:5192.
  2. Database: apply migrations (dotnet ef database update) or run the provided SQL script(s) in `backend/seed/` using SSMS.
  3. Frontend: open `frontend/employee-management` and run `npm install --legacy-peer-deps` then `ng serve` (or build & serve the `dist/` for production).
